const METHOD_COLORS: Record<string, string> = {

IMPORTANT: Since the app's `primary` color is green (#17c964 per Phase 1 D-07), POST cannot use `text-primary` (that would also be green). Use explicit Tailwind color classes instead:
- GET: `text-success` (green)
- POST: `text-blue-500` (blue)
- PUT: `text-warning` (orange)
- DELETE: `text-danger` (red)

**2. `RenameInput.tsx` — per D-05:**
Props: `{ currentName: string; onConfirm: (newName: string) => void; onCancel: () => void }`
- Renders a HeroUI `Input` (size="sm") with the current name pre-filled and selected.

**3. `DeleteModal.tsx` — per D-06:**

**4. `TreeContextMenu.tsx` — per D-03 and D-04:**
Props: `{ nodeType: 'collection' | 'folder' | 'request'; collectionSlug: string; parentPath: string[]; slug: string; nodeName: string; childCount: number }`
- Reads `contextMenuNodeId` and `contextMenuPosition` from `useCollectionStore`.
- Uses HeroUI `Dropdown`, `DropdownTrigger`, `DropdownMenu`, `DropdownItem`, `DropdownSection`.
- The trigger is a zero-size `<span>` positioned at `contextMenuPosition` via `style={{ position: 'fixed', left: x, top: y }}`.
- `isOpen` is true when `contextMenuNodeId` matches this node's unique ID (constructed as `collectionSlug/parentPath.join('/')/slug`).
- `onOpenChange(false)` calls `setContextMenu(null, null)`.

- Actions:
  - New Request: calls `createRequest(collectionSlug, parentPath, "New Request")` — for collection nodes, parentPath is []; for folder nodes, parentPath includes the folder slug.
  - New Folder: calls `createFolder(collectionSlug, parentPath, "New Folder")`
  - Rename: calls `setRenamingNode(nodeId)`
  - Duplicate: calls `duplicateRequest(collectionSlug, parentPath, slug)`
  - Delete: opens DeleteModal (use local state `showDeleteModal`)
- Add `window.addEventListener('blur', close)` per Pitfall 4 to close menu when window loses focus.
