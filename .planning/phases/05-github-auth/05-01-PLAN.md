**Step 4: Create `src-tauri/src/auth/token.rs`.**

Three functions using `tauri::AppHandle` and `tauri_plugin_secure_storage::SecureStorageExt`:
- `pub fn store_token(app: &tauri::AppHandle, token: &str) -> Result<(), String>` — calls `app.secure_storage().set_item(...)` with key `"github_token"`. Map errors to String.
- `pub fn load_token(app: &tauri::AppHandle) -> Result<Option<String>, String>` — calls `app.secure_storage().get_item(...)` with key `"github_token"`. Returns `Ok(None)` if not found (not an error).
- `pub fn clear_token(app: &tauri::AppHandle) -> Result<(), String>` — calls `app.secure_storage().remove_item(...)` with key `"github_token"`. Ignores "not found" errors.

Use `tauri_plugin_secure_storage::models::OptionsRequest` struct with `prefixed_key: Some("github_token".to_string())`.

**Step 5: Create `src-tauri/src/auth/device_flow.rs`.**

Types (all derive `Serialize, Deserialize, specta::Type, Clone`):
```rust
pub struct DeviceCodeResponse {
    pub device_code: String,
    pub user_code: String,
    pub verification_uri: String,
    pub expires_in: u32,
    pub interval: u64,
}
```

Functions:
- `pub async fn initiate_device_flow(client_id: &str) -> Result<DeviceCodeResponse, String>` — POST to `https://github.com/login/device/code` with form body `client_id={client_id}&scope=repo`, header `Accept: application/json`. Parse response as `DeviceCodeResponse`. Use `tauri_plugin_http::reqwest::Client::new()`.

- `pub async fn poll_for_token(client_id: &str, device_code: &str, initial_interval: u64) -> Result<String, String>` — Loop: sleep `interval` seconds, POST to `https://github.com/login/oauth/access_token` with form body `client_id`, `device_code`, `grant_type=urn:ietf:params:oauth:grant-type:device_code`, header `Accept: application/json`. Use `#[derive(Deserialize)] #[serde(untagged)] enum PollResponse { Success { access_token: String, ... }, Error { error: String } }`. On `authorization_pending` continue; on `slow_down` add 5 to interval permanently and continue; on `expired_token` return `Err("expired")`; on `access_denied` return `Err("denied")`; on success return `Ok(access_token)`.

The `client_id` value: Read from `option_env!("GITHUB_CLIENT_ID")` at compile time. If not set, fall back to a hardcoded development client ID constant `const DEV_CLIENT_ID: &str = "Ov23liXXXXXXXXXXXXXX";` (placeholder — user must register a GitHub OAuth App and set the env var). Define this constant in `device_flow.rs` and expose `pub fn get_client_id() -> &'static str` that returns `option_env!("GITHUB_CLIENT_ID").unwrap_or(DEV_CLIENT_ID)`.

Loop until empty page. Collect all into Vec. Sort by `owner.login` then `name`.

**Step 2: Create `src-tauri/src/workspace/registry.rs`.**

Type (derive `Serialize, Deserialize, specta::Type, Clone`):
```rust
pub struct WorkspaceEntry {
    pub id: String,               // UUID
    pub display_name: String,     // "repo-name" or "Local"
    pub github_repo_full_name: Option<String>,  // "owner/repo" — None for local
    pub clone_url: Option<String>,              // HTTPS URL — None for local
    pub local_path: String,       // absolute path to workspace dir
    pub is_local: bool,           // true only for permanent "Local" workspace
}
```

Functions:
- `pub fn load_registry(app: &tauri::AppHandle) -> Result<Vec<WorkspaceEntry>, String>` — Uses `tauri_plugin_store::StoreExt` to open `"dispatch-prefs.json"` store. Read key `"connected_workspaces"`. If missing or empty, call `bootstrap_local_workspace(app)` to create the "Local" entry and return it. Deserialize as `Vec<WorkspaceEntry>`.

- `pub fn save_registry(app: &tauri::AppHandle, entries: &[WorkspaceEntry]) -> Result<(), String>` — Serialize `entries` to JSON value, write to `"connected_workspaces"` key in `"dispatch-prefs.json"` store. Call `store.save()`.

- `fn bootstrap_local_workspace(app: &tauri::AppHandle) -> Result<Vec<WorkspaceEntry>, String>` — Get `app_data_dir`, read existing workspace IDs from `collections::io::get_workspace_ids(&app_data_dir)`. Use the first ID (the default workspace). Create a `WorkspaceEntry { id: first_id, display_name: "Local".to_string(), github_repo_full_name: None, clone_url: None, local_path: app_data_dir.join("workspaces").join(&first_id).to_string_lossy().to_string(), is_local: true }`. Save and return.

- `pub fn add_workspace(app: &tauri::AppHandle, entry: WorkspaceEntry) -> Result<(), String>` — Load registry, push new entry, save.

- `pub fn remove_workspace(app: &tauri::AppHandle, workspace_id: &str) -> Result<(), String>` — Load registry, retain entries where `id != workspace_id && !is_local`, save. (Cannot remove local workspace.)

**Step 3: Create `src-tauri/src/workspace/clone_ops.rs`.**

Functions:
- `pub fn clone_repo(clone_url: &str, local_path: &std::path::Path, token: &str) -> Result<(), String>` — Synchronous function (MUST be called from spawn_blocking). Create `git2::RemoteCallbacks::new()`, set credentials callback to `Cred::userpass_plaintext("oauth2", &token)`. Create `FetchOptions` with callbacks. Create `RepoBuilder::new()`, set `fetch_options`. Call `builder.clone(clone_url, local_path)`. Map git2::Error to String.

- `pub fn remove_clone(local_path: &std::path::Path) -> Result<(), String>` — Call `std::fs::remove_dir_all(local_path)`. Map io::Error to String. No-op if path doesn't exist.

**Step 4: Create `src-tauri/src/commands/workspace.rs`.**

IPC commands (all `#[tauri::command] #[specta::specta] pub async fn`):

- `list_workspaces(app: tauri::AppHandle) -> Result<Vec<WorkspaceEntry>, String>` — Call `registry::load_registry(&app)`.
