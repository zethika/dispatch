**Step 2: Add two new public functions to `src-tauri/src/collections/io.rs`.**

Add `pub fn reorder_node(parent_dir: &Path, slug: &str, new_index: usize) -> anyhow::Result<()>`:
- Read the manifest at `dir_manifest_path(parent_dir)`
- Check if `is_collection` (filename == `_collection.json`)
- If collection: read `CollectionManifest`, call `.order.retain(|s| s != slug)`, then `.order.insert(new_index.min(order.len()), slug.to_string())`, write manifest back
- If folder: same logic with `FolderManifest`
- Return `Ok(())`

Add `pub fn move_node(src_parent_dir: &Path, dst_parent_dir: &Path, workspace_dir: &Path, slug: &str, is_dir: bool, dst_index: Option<usize>) -> anyhow::Result<()>`:
- Call `assert_within(workspace_dir, src_parent_dir)?` and `assert_within(workspace_dir, dst_parent_dir)?`
- Compute `src_path`: if `is_dir` then `src_parent_dir.join(slug)` else `src_parent_dir.join(format!("{slug}.json"))`
- Compute `dst_path`: if `is_dir` then `dst_parent_dir.join(slug)` else `dst_parent_dir.join(format!("{slug}.json"))`
- Call `fs::rename(&src_path, &dst_path)?`
- Call `remove_from_order(&dir_manifest_path(src_parent_dir), slug)?`
- For dst manifest: read manifest, insert slug at `dst_index` (if `Some(i)` insert at `i.min(order.len())`; if `None` push to end), write manifest back
- Return `Ok(())`

**Step 3: Add two new Tauri commands in `src-tauri/src/commands/collections.rs`.**

```rust
#[tauri::command]
#[specta::specta]
pub fn reorder_node(
    workspace_id: String,
    collection_slug: String,
    parent_path: Vec<String>,
    slug: String,
    new_index: u32,
    app: tauri::AppHandle,
) -> Result<(), String> {
    let parent_dir = resolve_parent_dir(&app, &workspace_id, &collection_slug, &parent_path)?;
    io::reorder_node(&parent_dir, &slug, new_index as usize).map_err(|e| e.to_string())
}

#[tauri::command]
#[specta::specta]
pub fn move_node(
    workspace_id: String,
    src_collection_slug: String,
    src_parent_path: Vec<String>,
    dst_collection_slug: String,
    dst_parent_path: Vec<String>,
    slug: String,
    is_dir: bool,
    dst_index: Option<u32>,
    app: tauri::AppHandle,
) -> Result<(), String> {
    let ws_dir = workspace_dir(&app, &workspace_id)?;
    let src_dir = resolve_parent_dir(&app, &workspace_id, &src_collection_slug, &src_parent_path)?;
    let dst_dir = resolve_parent_dir(&app, &workspace_id, &dst_collection_slug, &dst_parent_path)?;
    io::move_node(&src_dir, &dst_dir, &ws_dir, &slug, is_dir, dst_index.map(|i| i as usize))
        .map_err(|e| e.to_string())
}
```

Note: Use `u32` not `usize` for index parameters because specta/Tauri IPC does not support `usize`. Cast to `usize` inside the function.
