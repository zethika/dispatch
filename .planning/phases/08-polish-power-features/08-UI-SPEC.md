## Spacing Scale

Declared values (must be multiples of 4). Scale follows existing codebase patterns (px-2/py-1 = 8px/4px, px-3/py-1.5 = 12px/6px nodes).

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps within tree nodes, inline badge padding |
| sm | 8px | Tree node vertical padding (py-1), icon-to-label gap (gap-1) |
| md | 16px | Tree node horizontal indent step per depth level, section padding |
| lg | 24px | Modal body padding, search result group spacing |
| xl | 32px | Modal header/footer spacing |
| 2xl | 48px | Full empty-state vertical centering gap |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Drop indicator line: 2px height (h-0.5), horizontally inset 8px on each side (mx-2) ŌĆö below the 4px floor by design for visual subtlety
- DnD activation constraint: 8px pointer distance before drag activates (not a spacing token; behavioral threshold per RESEARCH.md Pattern 1)
- Search modal: max-w-xl (576px) centered, uses HeroUI Modal default padding

**Source:** Existing codebase patterns (RequestNode px-2/py-1, CollectionTree px-3/py-2, TreeContextMenu py-1/px-3). RESEARCH.md Pattern 1 (drop indicator).

## Interaction Contracts

### Drag-and-Drop

- Drag activation: 8px pointer movement threshold (PointerSensor `activationConstraint: { distance: 8 }`)
- Drag start: dragged item renders at 40% opacity in-place; `DragOverlayItem` appears under cursor
- While dragging: `DropIndicator` appears above/below the item nearest to cursor midpoint
- Drop on folder: folder row shows `border-primary` ring when `isOver` ŌĆö releases item into folder as last child
- Drop on gap between items: `DropIndicator` line shows at that position ŌĆö releases item at that index
- Drop end: original position opacity restores; overlay disappears; `collectionStore.reorderNode()` or `collectionStore.moveNode()` fires; Rust IPC call commits order to disk

### cURL Import (paste auto-detect)

- Trigger: `onPaste` event on UrlBar input; check `pastedText.trimStart().startsWith('curl ')`
- On match: call `parseCurl(pastedText)`, populate `requestStore` fields (method, url, headers, body), suppress default paste, show sonner toast "cURL imported"
- On no match: default paste behavior ŌĆö text lands in URL field as normal
- On parse error: show sonner toast "Could not parse cURL command" (warning, not error)
- Context menu import: opens `CurlImportModal`; on confirm calls same `parseCurl()` path

### cURL Export (Cmd+Shift+C)

- Action: build cURL string using `buildCurlString()` with `environmentStore.activeEnvVariables`, copy to clipboard via `navigator.clipboard.writeText()`
- Feedback: sonner toast "Copied as cURL" (brief, no action button)
- No modal ŌĆö fire-and-forget

### Cmd+K Search

- Open: sets `uiStore.searchOpen = true`; modal appears with framer-motion fade-in (duration 150ms, ease out)
- Input: autofocused on open; placeholder "Search requests, collections..."
- Results: update on every keystroke via Fuse.js (threshold 0.4, keys weighted name > url > collectionName)
- Results layout: grouped by collection; each group has a `text-xs font-semibold uppercase text-default-500` header
- Result row: request name (text-sm) on top, breadcrumb below (text-xs text-default-400 truncate)
- Keyboard navigation: arrow keys move selection; selected row has `bg-primary/10` highlight
- Enter: navigate to selected request (calls `collectionStore.setActiveRequest()`, closes modal)
- Escape: close modal
- Empty results state: "No results for '{query}'" ŌĆö text-sm text-default-500, vertically centered in results area
- No query state: "Type to search requests and collections" ŌĆö text-sm text-default-400

### Shortcut Flash Feedback (D-12)

- On shortcut fire: add `ring-2 ring-primary ring-offset-1 rounded` to target element for 300ms, then remove
- Timing: 0ms delay, 300ms duration, no easing (instant on, instant off)
- Implementation: `setTimeout` on a CSS class toggle ŌĆö no framer-motion needed for this
- Targets by shortcut:
  - Cmd+Enter: Send button in UrlBar
  - Cmd+N: sidebar new-request icon or focused CollectionNode
  - Cmd+Shift+N: sidebar area
  - Cmd+K: search icon in TopBar
  - Cmd+E: environment selector in TopBar
  - Cmd+Shift+C: copy icon or UrlBar area
  - Cmd+W: active request tab label
  - Cmd+S: SyncStatusChip

### Keyboard Shortcuts (Global)

All registered in single `useEffect` in `App.tsx`. All fire regardless of focus (D-10).

| Shortcut | Target Action |
|----------|--------------|
| Cmd+Enter | `requestStore.sendRequest()` |
| Cmd+N | `collectionStore.createRequest()` on active collection |
| Cmd+Shift+N | `collectionStore.createCollection()` |
| Cmd+K | `uiStore.setSearchOpen(true)` |
| Cmd+E | `environmentStore.openSelector()` (opens environment dropdown) |
| Cmd+Shift+C | `buildCurlString()` + clipboard + toast |
| Cmd+W | `collectionStore.clearActiveRequest()` |
| Cmd+S | `syncStore.triggerPush()` |
| Cmd+/ | `uiStore.setCheatsheetOpen(true)` |
