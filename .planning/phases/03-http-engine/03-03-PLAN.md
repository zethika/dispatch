    **1. Create `src/features/http/JsonViewer.tsx` (per D-10, D-11):**

    **Tokenizer function `tokenizeJson(json: string): Token[]`:**
    - Define `TokenType = 'key' | 'string' | 'number' | 'boolean' | 'null' | 'punctuation' | 'whitespace'`
    - Define `Token = { type: TokenType; value: string }`
    - Use regex: `("(?:[^"\\]|\\.)*")|(\b-?\d+(?:\.\d+)?(?:[eE][+-]?\d+)?\b)|(true|false)|(null)|([{}\[\],:])|(\s+)`
    - Scan with `TOKEN_REGEX.exec(json)` in a while loop, collecting tokens
    - Post-process: mark string tokens followed by `:` (skipping whitespace) as `'key'` type
    - Handle unmatched characters as `'punctuation'` type

    **Token color map (Tailwind classes):**
    ```
    key: 'text-blue-400'
    string: 'text-green-400'
    number: 'text-yellow-400'
    boolean: 'text-purple-400'
    null: 'text-gray-500'
    punctuation: 'text-default-500'
    whitespace: '' (no class)
    ```

    **JsonViewer component:**
    - Props: `{ body: string }` — the raw response body string
    - Try `JSON.parse(body)` then `JSON.stringify(parsed, null, 2)` to pretty-print. If parse throws, set a flag `isJson = false` and render raw text fallback (per D-12, RESP-05).
    - **Large response guard (Pitfall 5):** If `body.length > 102400` (100KB), skip tokenizer and render raw text in a `<pre>` with a note: "Response body is large (>100KB). Showing raw text." Use `text-default-400 text-xs italic mb-2` for the note.
    - For JSON responses (isJson=true, under 100KB): run `tokenizeJson(formattedJson)` and map tokens to `<span>` elements with the color class from TOKEN_COLORS map. Wrap in `<pre className="font-mono text-sm whitespace-pre-wrap break-words">`.
    - For non-JSON responses: render `<pre className="font-mono text-sm whitespace-pre-wrap break-words text-foreground">{body}</pre>`.
    - **Copy button (per D-10):** Position a HeroUI `Button` with `isIconOnly`, `size="sm"`, `variant="light"` in the top-right corner (absolute positioned within a relative container). OnPress: copy raw JSON (not formatted) to clipboard via `navigator.clipboard.writeText(body)`. Show brief feedback: swap button text from "Copy" icon to "Copied!" for 2 seconds using local state.
    - Export default `JsonViewer`.

Default to empty string for unknown codes.
    - **Duration:** `<span className="text-default-500 text-sm">{durationMs} ms</span>`
